2	Для чего предназначен объект Date?
Для вывода даты ивремени

3	Что называют массивом?
Массив - это структура данных, 
хранящая набор значений (элементов массива), 
идентифицируемых по индексу

4	Для чего предназначены массивы?
Для хранения значений или массивов

5	Каким образом осуществляется доступ к массиву?
myArray= [50, "length", "width", "height"];
myArray[2];

6	Каким образом производится установка размера массива?
устанавливается динамически. Если написать myArray[99]= "xyz", 
размер массива будет установлен 100 элементов.

7	Для чего предназначен объект Math?
Для выполнения математических расчётов

8	Перечислите методы объекта Date.

newDate = new Date();
